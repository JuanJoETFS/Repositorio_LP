🎓 Condicionales en Python: Expresión if

📘 ¿Qué es un condicional?

Los condicionales permiten tomar decisiones en el programa. La estructura básica evalúa si una condición es verdadera y ejecuta el código correspondiente.

🧱 Sintaxis básica

if condición:
    # bloque de código si es verdadera

🔎 Ejemplo práctico

edad = int(input("¿Cuál es tu edad?: "))
invitacion = input("¿Tienes invitación? (s/n): ")

if edad >= 18:
    if invitacion.lower() == 's':
        print("🎉 ¡Bienvenido al evento!")
    else:
        print("🚫 Necesitas una invitación para entrar.")
else:
    print("🚫 Lo siento, debes ser mayor de edad.")

🧠 Funcionalidades del if

  • if simple: Evalúa una sola condición.
  • if-else: Ejecuta una alternativa si la condición es falsa.
  • if-elif-else: Múltiples condiciones evaluadas en orden.
  • Anidado: Un if dentro de otro.
  • Compuesto: Uso de and, or, not.
